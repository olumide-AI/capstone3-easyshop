🎤 EasyShop: My Code Journey

A full-featured Java-based e-commerce platform with dynamic filters, login, and a shopping cart.

About the Project 🚀

EasyShop is an educational and practical e-commerce app I built to practice real-world coding standards. It simulates an online shopping system using modular Java services, CRUD operations, and database integration.

Key Features ✨

  • 🛍️ Manage products: add, update, delete, list
  • 👤 Customer registration and profile management
  • 🧾 Create and process orders
  • ✅ Console and web-based UI with dynamic filters
  • 🔎 Search products by category, price, and color

Architecture Overview 🏗️

  • Presentation: Console-based UI & dynamic frontend
  • Service Layer: Business logic handling products, customers, and orders
  • DAO Layer: Data access for CRUD operations
  • Model Layer: Java classes representing domain entities
  • 🔌 Designed for separation of concerns and easy testing

Technologies & Tools 🛠️

  • Languages: Java 17+, HTML, CSS, JavaScript
  • Libraries & Frameworks: Bootstrap, Mustache.js, Axios
  • Testing: JUnit 5, Mockito
  • Database: MySQL
  • Build & IDE: Maven, IntelliJ IDEA

@Test
public void getAllCategoriesTest() {
    List expectedCategoryList = new ArrayList<>();
    expectedCategoryList.add(new Category(1, "charger", "A phone charger"));
    expectedCategoryList.add(new Category(2, "iphone", "An iPhone"));

    when(categoryDao.getAllCategories()).thenReturn(expectedCategoryList);

    List categoryList = controller.getAll();

    assertThat(categoryList)
        .isNotEmpty()
        .isEqualTo(expectedCategoryList)
        .hasSize(2);
}
